System Server Requirements

php.ini

  1. upload_max_filesize

...Sets a maximum amount of memory in bytes that a script is allowed to allocate. Resizing a 3000 x 2000 pixel image to 300 x 200 may take up to 32MB memory.

  1. php Extensions and version :

  2. PHP >= 7.0

  3. Fileinfo Extension
  4. GD Library (>=2.0) … or …
  5. Imagick PHP extension (>=6.5.7)

Image Optimization tools (https://github.com/spatie/image-optimizer)

  1. Here's how to install all the optimizers on Ubuntu:
  2. sudo apt-get install jpegoptim
  3. sudo apt-get install optipng
  4. sudo apt-get install pngquant
  5. sudo npm install -g svgo

  6. sudo apt-get install gifsicle

  7. And here's how to install the binaries on MacOS (using Homebrew):

  8. brew install jpegoptim
  9. brew install optipng
  10. brew install pngquant
  11. brew install svgo
  12. brew install gifsicle

Other

  • brew install --with-openssl curl
  • brew install --with-homebrew-curl --with-apache php71
  • brew install php71-mcrypt php71-imagick
